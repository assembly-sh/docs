---
title: "Update a Project"
description: "Update a Project for the current user."
---

### Parameters

<ResponseField name="name" type="string">
  The name of the project.
</ResponseField>

<ResponseField name="description" type="string">
  The description of the project.
</ResponseField>

<ResponseField name="organization_id" type="string">
  The organization ID of the project.
</ResponseField>

### Response

Returns the updated Project object if successful. Otherwise, this call raises an [error](../errors).


<RequestExample>
```bash POST /v1/projects/:id
curl 'https://api.assembly.sh/v1/projects/fca28702-2ce9-45e0-ad21-e9320aa08731' \
-u '<token>:' \
-d 'name=My amazing project' \
-d 'description=This is my amazing project' \
```
</RequestExample>

<ResponseExample>
```json Response
{
  "id": "fca28702-2ce9-45e0-ad21-e9320aa08731",
  "object": "project",
  "organization_id": "752268f6-102a-40c8-b726-0a5abe891083",
  "name": "My amazing project",
  "description": "This is my amazing project",
  "created_at": "2023-09-18T14:33:27.138043176Z",
  "updated_at": "2023-09-18T15:26:12.358243464Z"
}
```
</ResponseExample>
