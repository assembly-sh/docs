---
title: "Project endpoints"
description: "Project endpoints allow you to create, list, get and delete Projects."
---

A project is a collection of services and their associated resources.

You can create as many projects as you like, and each project belongs to an organization.


<RequestExample>
```js ENDPOINTS
POST /v1/projects
GET /v1/projects 
GET /v1/projects/:id
POST /v1/projects/:id
DELETE /v1/projects/:id
```
</RequestExample>
