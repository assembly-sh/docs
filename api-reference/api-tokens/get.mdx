---
title: "Retrieve an ApiToken"
description: "Retrieve an ApiToken that has previously been created."
---

### Parameters

No parameters.

### Response

Returns the ApiToken object for a valid identifier. Otherwise, this call raises an [error](../errors).


<RequestExample>
```bash GET /v1/api_tokens/:id
curl 'https://api.assembly.sh/v1/api_tokens/47ea4149-1922-4c1c-afba-924fcde0f3a8' \
-u '<token>:'
```
</RequestExample>

<ResponseExample>
