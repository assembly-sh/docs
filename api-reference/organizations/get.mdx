---
title: "Retrieve an Organization"
description: "Retrieve an Organization that has previously been created."
---

### Parameters

No parameters.

### Response

Returns the Organization object for a valid identifier. Otherwise, this call raises an [error](../errors).


<RequestExample>
```bash GET /v1/organizations/:id
curl 'https://api.assembly.sh/v1/organizations/752268f6-102a-40c8-b726-0a5abe891083' \
-u '<token>:'
```
</RequestExample>
