---
title: "Create an Organization"
description: "Creates an Organization for the current user."
---

### Parameters

<ResponseField name="name" type="string" required>
The name of the organization.
</ResponseField>

<ResponseField name="slug" type="string" required>
The slug of the organization.
</ResponseField>

<ResponseField name="type" type="string" required>
The type of the organization. Can be `pro` or `free`.
</ResponseField>

### Response

Returns the created Organization object if successful. Otherwise, this call raises an [error](../errors).


<RequestExample>
```bash POST /v1/organizations
curl 'https://api.assembly.sh/v1/organizations' \
-u '<token>:' \
-d 'name=My Company' \
-d 'slug=my-company' \
-d 'type=pro'
```
</RequestExample>
