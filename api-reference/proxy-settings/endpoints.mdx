A ProxySettings object represents a revision of a proxy configuration of a Project.


<RequestExample>
```js ENDPOINTS
GET /v1/projects/{id}/proxy-settings
POST /v1/projects/{id}/proxy-settings
POST /v1/projects/{id}/proxy-settings/:revision
```
</RequestExample>
