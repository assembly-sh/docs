---
title: "Delete a Proxy"
description: "Delete a Proxy that has previously been created."
---

### Parameters

No parameters.

### Response

Returns an object with a deleted parameter on success. If the Proxy ID does not exist, this call returns an [error](../errors).

Deleted Proxys cannot be recovered and won't be usable anymore.

<RequestExample>
```bash DELETE /v1/proxies/:id
curl -X DELETE 'https://api.assembly.sh/v1/proxies/7fca28702-2ce9-45e0-ad21-e9320aa08731' \
-u '<token>:'
```
</RequestExample>

<ResponseExample>
```json Response
{
  "id": "fca28702-2ce9-45e0-ad21-e9320aa08731",
  "object": "proxy",
  "deleted": true
}
```
</ResponseExample>
