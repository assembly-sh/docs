---
title: "Create a Proxy"
description: "Creates a Proxy for a Project inside an Organization."
---

### Parameters

<ResponseField name="name" type="string" required>
The name of the proxy.
</ResponseField>

<ResponseField name="slug" type="string" required>
The slug of the proxy.
</ResponseField>

<ResponseField name="type" type="string" required>
The type of the proxy. Can be `pro` or `free`.
</ResponseField>

### Response

Returns the created Proxy object if successful. Otherwise, this call raises an [error](../errors).


<RequestExample>
```bash POST /v1/proxies
curl 'https://api.assembly.sh/v1/proxies' \
-u '<token>:' \
-d 'name=My Proxy' \
-d 'description=My Proxy description' \
-d 'organization_id=752268f6-102a-40c8-b726-0a5abe891083' \
```
</RequestExample>
