---
title: "Retrieve a Proxy"
description: "Retrieve a Proxy that has previously been created."
---

### Parameters

No parameters.

### Response

Returns the Proxy object for a valid identifier. Otherwise, this call raises an [error](../errors).


<RequestExample>
```bash GET /v1/proxies/:id
curl 'https://api.assembly.sh/v1/proxies/fca28702-2ce9-45e0-ad21-e9320aa08731' \
-u '<token>:'
```
</RequestExample>

<ResponseExample>
```json Response
{
  "id": "fca28702-2ce9-45e0-ad21-e9320aa08731",
  "object": "proxy",
  "organization_id": "752268f6-102a-40c8-b726-0a5abe891083",
  "name": "My Projet",
  "description": "My Proxy description",
  "created_at": "2023-09-18T14:33:27.138043176Z",
  "updated_at": "2023-09-18T14:33:27.138043176Z"
}
```
</ResponseExample>
